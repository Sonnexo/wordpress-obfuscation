# wordpress-obfuscation
This is a plugin for wordpress based on the plugin "Email Address Obfuscation" by [Neotrendy](https://profiles.wordpress.org/neotrendy/).
Here is the link to the [plugin page](https://wordpress.org/plugins/email-address-obfuscation/) and here the direct link to the [source code](https://plugins.trac.wordpress.org/browser/email-address-obfuscation/).

# Functionality
To obfuscate e-mail-addresses, use a shortcode with **[obfuscate_email email="recipient@domain.xx" link=true]**, while for ordinary text, use **[obfuscate_data data="secret text"]**.
